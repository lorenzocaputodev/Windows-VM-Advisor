Windows-VM-Advisor - VM Profiles
================================

Use these settings as a practical starting point. Increase them only if needed.

Best choices:
- Linux Mint [Recommended]
   Hypervisor: VMware Workstation
   Suggested setup: 2 vCPU, 4096 MB RAM, 48 GB disk, UEFI, Secure Boot Disabled, TPM No, Network NAT
   Save VM files on: D:

- Ubuntu LTS [Recommended]
   Hypervisor: VMware Workstation
   Suggested setup: 2 vCPU, 4096 MB RAM, 48 GB disk, UEFI, Secure Boot Disabled, TPM No, Network NAT
   Save VM files on: D:

- Windows 10 [Recommended]
   Hypervisor: VMware Workstation
   Suggested setup: 2 vCPU, 6144 MB RAM, 80 GB disk, UEFI, Secure Boot Disabled, TPM No, Network NAT
   Save VM files on: D:
